# Aplikasi Pendataan Buku
# Deskripsi Singkat
Project ini adalah aplikasi sederhana berbasis Command Line Interface (CLI) yang dibuat menggunakan Python dasar.

Aplikasi ini memungkinkan pengguna untuk menyimpan dan membaca buku secara sederhana melalui terminal. Semua data buku disimpan ke dalam file data.txt, sehingga tetap tersimpan walaupun program ditutup.

Project ini dibuat sebagai Proyek Akhir mata pelajaran Pemrograman Dasar – Python.

# Tujuan
Tujuan dari project ini adalah untuk membuktikan bahwa saya memahami:

- Cara membuat program berbasis menu

- Cara menggunakan fungsi dalam Python

- Penggunaan list dan dictionary

- Perulangan (loop)

- Percabangan (if-else)

- Error handling menggunakan try-except

- Penyimpanan dan pembacaan file (.txt)

Program ini dibuat tanpa menggunakan GUI atau library tambahan, sesuai ketentuan proyek.

# Fitur Utama

Fitur yang Tersedia

Aplikasi ini memiliki beberapa fitur utama:

- Melihat daftar buku yang tersedia

- Menambahkan buku baru

- Membaca isi buku yang dipilih

- Menghapus buku

- Keluar dari program

Program akan terus berjalan sampai pengguna memilih menu keluar.

# Cara Kerja Program

- Data buku disimpan dalam bentuk list.

- Setiap buku disimpan dalam bentuk dictionary yang berisi:

   - Judul

   - Penulis

- Isi

- Saat program dijalankan, data akan dimuat dari data.txt.

- Setiap ada perubahan (tambah/hapus buku), data langsung disimpan kembali ke file.

Program juga menggunakan try-except untuk mencegah error saat pengguna salah memasukkan input.

# Cara Menjalankan Program

1.Pastikan Python sudah terinstall di komputer.

2.Pastikan file berikut berada dalam satu folder:

 - main.py

 - data.txt

3.Buka terminal atau Command Prompt.

4.Jalankan perintah:

     python main.py


5.Pilih menu sesuai kebutuhan.

# Struktur File
main.py   → File utama program

data.txt  → File penyimpanan data buku

README.md → Dokumentasi project
